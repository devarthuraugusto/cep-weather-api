# CEP Weather API

Projeto em Python que:

- Recebe um CEP do usuário
- Busca o endereço usando uma API de CEP
- Obtém a cidade
- Consulta a temperatura da cidade usando uma API de clima
- Exibe as informações organizadas

## Tecnologias

- Python
- Requests
- APIs REST
- JSON

## Como rodar

1. Instale as dependências

pip install -r requirements.txt

2. Execute o programa

python main.py
